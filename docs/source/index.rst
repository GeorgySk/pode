Welcome to pode's documentation!
===================================

**Pode** is a Python library that implements the algorithm of [Hert, S.
and Lumelsky, V., 1998](https://www.worldscientific.com/doi/abs/10.1142/S0218195998000230) for a polygon decomposition into separate parts
depending on the area requirements..
